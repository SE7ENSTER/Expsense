# ExpenseFlow

A modern web app draft for travel expense reports.

## Features

- Add and edit expense items
- Select category, currency and exchange rate
- Upload a receipt image per item
- Preview receipts
- Export one PDF with summary and receipt attachment pages

## Run

```bash
npm install
npm run dev
```
